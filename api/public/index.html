API Endpoints

GET /images
Get all images or search with query parameter
GET /images?name_like=sunset
GET /images/:id
Get a specific image by ID
GET /images/123e4567-e89b-12d3-a456-426614174000
POST /images
Create a new image
POST /images Content-Type: application/json { "name": "sunset.jpg", "url": "data:image/jpeg;base64,/9j/4AAQ...", "size": 1024576, "type": "image/jpeg" }
PATCH /images/:id
Update an existing image (rename)
PATCH /images/123e4567-e89b-12d3-a456-426614174000 Content-Type: application/json { "name": "new-name.jpg" }
DELETE /images/:id
Delete an image by ID
DELETE /images/123e4567-e89b-12d3-a456-426614174000
GET /images/stats
Get database statistics
GET /images/stats
GET /health
Health check endpoint
GET /health
